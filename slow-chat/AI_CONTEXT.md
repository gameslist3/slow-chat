# Project: Slowchat / Antigravity

## Tech Stack
- Frontend: Vite + React
- Language: TypeScript + some JSX
- Styling: Tailwind CSS
- Hosting: Firebase Hosting
- Auth: Firebase Auth
- Database: Firestore
- Storage: Cloudinary (uploads)

## Repo
- GitHub: https://github.com/gameslist3/slow-chat

## Build / Deploy
- Local dev: npm run dev
- Build: npm run build
- Deploy: GitHub Actions -> Firebase Hosting

## Important rules
- Must NOT break npm run build
- Keep UI in modern 2026 AI-app style (dark default + light mode option)
- User messages on right, others left
- Avoid adding heavy libraries unless needed
